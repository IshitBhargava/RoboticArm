# RoboticArm

Build a robotic arm easily and control it using a Mobile App!

# Things Needed:

<pre>To make this, you need:
1x Arduino NANO
1x HC-05 OR HC-06
1x Breadboard (Or you can use the PCB)
3x SG90 Servo
3x MG995 Servo
1x USB Cable
4x Buttons (Optional)
3x 3 milimeter LEDs
</pre>

__________________________________________________________________________________________

# Connections:

**Servo 1 (Grab):**
<pre>VCC: Vin
GND: GND
SIGNAL: D5
</pre>


**Servo 2 (Wrist Pitch):**
<pre>VCC: Vin
GND: GND
SIGNAL: D6
</pre>


**Servo 3 (Wrist Roll):**
<pre>VCC: Vin
GND: GND
SIGNAL: D3
</pre>


**Servo 4 (Elbow):**
<pre>VCC: Vin
GND: GND
SIGNAL: D9
</pre>


**Servo 5 (Shoulder):**
<pre>VCC: Vin
GND: GND
SIGNAL: D10
</pre>


**Servo 6 (Base):**
<pre>VCC: Vin
GND: GND
SIGNAL: D11
</pre>
